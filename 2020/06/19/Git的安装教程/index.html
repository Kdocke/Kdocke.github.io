Git 的安装教程

教程 Git 安装 2020/06/19

一、下载

网址：git 官网 或 淘宝镜像

二、安装

1. 双击安装程序，点击 Next

2. 更改安装路径，建议 D 盘，点击 Next

3. 选择安装组件，可以直接 Next

4. 直接 Next，创建开始菜单

5. 选择编辑方式，默认使用 Vim，点击 Next

6. 使用方式，三种，默认选择第二种

7. 选择 OpenSSL 远程连接方式，Next

8. 配置换行格式，默认，Next

9. 配置终端，默认，Next

10. 默认，Next

11. 默认配置，Next

12. 安装

13. 安装完成

三、验证

命令行输入 git --version
